Introductie

Bij dit project heb ik een simpele 2d platform game gemaakt, grotendeels in JavaScript en door middel van het canvas element. Hierbij kwamen genoeg complexe functies in voor en ook in herhaling, waardoor ik mijn kennis in JavaScript goed kon ontwikkelen.

Uitdaging

De meest uitdagende functie van de game was de collisions, wat zorgt ervoor dat de speler op de grond blijft staan en niet door de platform heen valt. Dit was lastig omdat de positie van de speler niet altijd precies op de locatie van de platforms uitkwam, dus om dit op te lossen heb ik een kleine marge aan de platforms toegevoegd waardoor het wel werkte.

Resultaat

Het resultaat van dit project vindt ik goed gelukt, ik heb namelijk de orginele schets goed kunnen uitwerken en ik heb veel kennis opgedaan over JavaScript.Ik denk dat bepaalde delen van de code wel wat efficiënter en eenvoudiger hadden gekund. Maar over het algemeen was het project een leuke en effectieve manier om mijn kennis van JavaScript te verbeteren. Uiteindelijk voel ik me nu een stuk zelfverzekerder in het toepassen van JavaScript.
